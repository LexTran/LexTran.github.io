Predicting Physical Properties of Complex Carbonate Reservoirs based on Deep Learning
Main developer. Conducting analysis and preprocessing of three-dimensional seismic quality data, developing an intelligent porosity prediction algorithm model, and utilizing the trained model for reservoir porosity prediction.

Seisimic Impedance Inversion Prediction based on Deep Learning
Main developer. By combining seismic attribute dimensions and logging data, a siamese DenseNet was constructed with seismic attribute dimensions as input and logging density data as output.

March. 2023 -- June.2023

Evaluation System for Key Factors Affecting Wellbore Leakage
Main developer. To analyze the impact of different gas field characteristics on well leakage prediction performance, an Artificial Neural Network (ANN) was trained to compare and evaluate the influence of various gas field features on well leakage prediction. Additionally, a GUI interface was developed using PyQT.

Oct. 2022 -- Dec.2022

Gas Field Wellbore Leakage Prediction System
Main developer. In order to perform intelligent predictive analysis before wellbore leakage occurs in oil wells, six types of kernel functions of Support Vector Machines (SVM) and Artificial Neural Networks (ANN) were trained using Principal Component Analysis (PCA) for gas field wellbore leakage prediction. Additionally, a GUI interface was developed using PyQT.

Oct. 2022 -- Dec.2022

Pocket Gardener
Main developer. The project is built on the CC2530 development board, integrating functions such as a temperature and humidity sensor and a wireless Bluetooth module, and has developed a smart remote-controlled flowerpot. The project technology involves embedded C language, CC2530 development board circuit principles, and sensor communication principles.
